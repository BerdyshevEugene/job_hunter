## Installation

clone the repository and go to it on the command line:
```sh
https://github.com/BerdyshevEugene/job_hunter.git
```

create and activate virtual enviroment:
```sh
python3 -m venv venv
source env/bin/activate
python3 -m pip install --upgrade pip
```
